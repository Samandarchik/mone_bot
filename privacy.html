1. Общие положения

Настоящая Политика конфиденциальности (далее — «Политика») определяет порядок сбора, использования, хранения и защиты персональных данных пользователей мобильного приложения Hr System (идентификатор: uz.uzaidev.hr), а также связанного веб-сервиса и Telegram-бота (далее совместно — «Сервис»).

Сервис предназначен для автоматизации процессов подбора персонала (HR) и управления кандидатами на вакансии.

Используя Сервис, вы подтверждаете, что ознакомились с данной Политикой и согласны с условиями обработки ваших персональных данных.

Оператор данных: Индивидуальный разработчик — @uzaidev. Связь: https://t.me/uzaidev

3. Цели обработки данных

Персональные данные обрабатываются исключительно для следующих целей:

  • Подбор персонала — прием и обработка анкет кандидатов, оценка их соответствия вакансиям
  • Организация интервью — планирование собеседований, отправка информации о месте и времени проведения
  • Управление HR-процессами — ведение базы кандидатов, отслеживание статуса рассмотрения заявок
  • Авторизация и безопасность — идентификация сотрудников HR, контроль доступа к системе
  • Коммуникация — отправка уведомлений кандидатам через Telegram (приглашение на интервью, отправка локации филиала)
  • Аналитика — формирование внутренней статистики по заявкам и интервью для улучшения процесса найма

Мы не используем ваши данные для рекламных рассылок, продажи третьим лицам или иных целей, не связанных с процессом трудоустройства.

4. Хранение и защита данных

4.1. Где хранятся данные

  • Сервер — данные хранятся в базе данных на защищенном сервере с HTTPS-шифрованием
  • Мобильное устройство — на устройстве сотрудника HR хранится только токен авторизации в локальном хранилище (SharedPreferences)
  • Фотографии — загруженные фотографии кандидатов хранятся в директории сервера с ограниченным доступом

4.2. Меры безопасности

  • Шифрование паролей — пароли хранятся в виде bcrypt-хешей и не могут быть восстановлены в исходном виде
  • Токены сессий — генерируются криптографически безопасным генератором случайных чисел (32 байта / 64 символа)
  • HTTPS — все соединения между приложением и сервером шифруются протоколом TLS
  • Ролевой доступ — разграничение прав доступа: обычные администраторы и супер-администраторы имеют разные уровни полномочий
  • Проверка активности — неактивные учетные записи блокируются и не могут получить доступ к системе

4.3. Сроки хранения

Персональные данные кандидатов хранятся в системе до момента их удаления администратором HR. Данные учетных записей сотрудников хранятся на протяжении периода их работы в организации.

5. Передача данных третьим лицам

Мы не продаем, не передаем и не раскрываем ваши персональные данные третьим лицам, за исключением следующих случаев:

  • Telegram Bot API — для отправки уведомлений кандидатам (подробнее в разделе 6)
  • Требования законодательства — если раскрытие данных требуется в соответствии с действующим законодательством Республики Узбекистан, по запросу уполномоченных государственных органов

6. Использование Telegram Bot API

Сервис интегрирован с Telegram Bot API для следующих функций:

  • Прием анкет кандидатов через Telegram-бота
  • Отправка уведомлений о статусе заявки
  • Отправка информации об интервью (дата, время, локация филиала)

При взаимодействии с Telegram-ботом, Telegram передает нам ваш User ID и username. Эти данные используются исключительно для идентификации и связи с кандидатом в рамках процесса найма.

Обработка данных со стороны Telegram регулируется Политикой конфиденциальности Telegram.

7. Права пользователей

Вы имеете следующие права в отношении своих персональных данных:

  • Право на доступ — вы можете запросить информацию о том, какие ваши данные хранятся в системе
  • Право на исправление — вы можете потребовать исправления неточных или неполных данных
  • Право на удаление — вы можете потребовать полного удаления ваших данных из системы
  • Право на отзыв согласия — вы можете в любой момент отозвать согласие на обработку данных

Для реализации своих прав свяжитесь с нами через Telegram: @uzaidev.

Мы обязуемся рассмотреть ваш запрос в течение 10 рабочих дней с момента получения и предпринять необходимые действия.

8. Файлы cookie и локальное хранилище

Мобильное приложение не использует файлы cookie. Для обеспечения функции автоматического входа в систему на устройстве сохраняется только токен авторизации через механизм SharedPreferences (Android) и UserDefaults (iOS).

Веб-версия сервиса может использовать стандартные механизмы браузера для хранения данных сессии.

При выходе из аккаунта (logout) токен удаляется как с устройства, так и с сервера.

9. Защита данных несовершеннолетних

Сервис предназначен для лиц, достигших 16 лет. Мы сознательно не собираем персональные данные лиц младше указанного возраста. Если вам стало известно, что несовершеннолетний предоставил нам свои данные без согласия родителей или законных представителей, пожалуйста, свяжитесь с нами для их удаления.

10. Изменения в политике конфиденциальности

Мы оставляем за собой право вносить изменения в настоящую Политику. В случае существенных изменений мы уведомим пользователей путем обновления даты последнего изменения в верхней части данного документа.

Рекомендуем периодически просматривать данную страницу для ознакомления с актуальной версией Политики.
